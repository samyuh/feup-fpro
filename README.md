# Game for FPRO

Simple game made in python, with module PyGame.

### Prerequisites

PyGame installed

    try one of the following commands under the cmd:
    ```
    pip install pygame
    
    sudo apt-get install python3-pygame
    
    ```

## How to Run

 Directly from your Python interpreter, run the file "Bubble.py".
